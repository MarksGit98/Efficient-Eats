Rank Menu Items by Nutrition


Below is a table containing menu items from a specific restaurant or all restaurants available on this site along with the Caloric content, Macronutrients, and Efficiency (percentage of total calories from protein) of each item.

Features / How to use:

  • Select the restaurant you wish to browse the menu from (the "All Restaurants" option may be laggy)
  • Check the box next to "Exclude Drinks" to remove all drinks from the table
  • Type in a minimum and maximum range in the calories and 3 macronutrient fields to only display items that fit within that range
  • Click on the labels on the table to filter items by that category
  • Display more entries by modifying the number in the "Show _ Entries" dropdown list
  • Use the search field to find a specific item or set of items by name on the table
  • Click on the names of items on the table to see that item's full nutritional information (nutrition label and macronutrient breakdown)
  • Navigate all of the items on the table with the Previous and Next buttons below the table to see all of the pages
